立即安装

  1. 下载扩展文件： 下载最新版本
  2. 打开 Edge 浏览器，访问 edge://extensions/
  3. 开启右上角的"开发人员模式"
  4. 点击"加载解压缩的扩展"
  5. 选择解压后的扩展文件夹
